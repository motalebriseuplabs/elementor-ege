== Description ==

Elementor edge is a comprehensive addon for Elementor Page Builder that extends its functionality with custom widgets, controls, and advanced features. Built following WordPress.org coding standards and Elementor best practices, this plugin is designed for professional developers and seamless WordPress.org submission.

**Key Features:**

* **Custom Widgets Collection** - Professional widgets including progress bars, counters, testimonials, and more
* **Advanced Controls** - Custom Elementor controls for enhanced design capabilities
* **Performance Optimized** - Separate admin and frontend asset loading for optimal performance
* **WordPress.org Compliant** - Follows all WordPress coding standards and best practices
* **Translation Ready** - Full internationalization support with proper text domains
* **Admin Interface** - Comprehensive settings panel with widget management
* **OOP Architecture** - Modern PHP object-oriented programming with proper namespacing

**Included Widgets:**

* **Hello Widget** - Demonstration widget with interactive elements
* **Progress Bar** - Animated progress indicators with customizable styling
* **Counter** - Animated number counters with scroll-triggered animations
* **Testimonials** - Rotating testimonial slider with navigation
* **Image Comparison** - Before/after image comparison with draggable slider

**Custom Controls:**

* **Query Control** - Advanced post/page selection with dynamic options
* **Image Choose Control** - Visual layout selection with image previews
* **Gradient Control** - Advanced gradient picker with multiple color stops

== Installation ==

**Automatic Installation:**

1. Log in to your WordPress admin dashboard
2. Navigate to Plugins → Add New
3. Search for "Elementor edge"
4. Click "Install Now" and then "Activate"
5. Go to Elementor edge in your admin menu to configure settings

**Manual Installation:**

1. Download the plugin from WordPress.org
2. Upload the `elementor-edge` folder to `/wp-content/plugins/`
3. Activate the plugin through the 'Plugins' scredgen in WordPress
4. Navigate to Elementor edge in your admin menu to configure settings

**After Installation:**

1. Make sure Elementor is installed and activated
2. Go to Elementor edge → Settings to configure the plugin
3. Enable the widgets you want to use
4. Start using the new widgets in Elementor editor

== Frequently Asked Questions ==

= What is the minimum Elementor version required? =

Elementor edge requires Elementor 3.0.0 or higher for optimal compatibility and to access the latest Elementor features.

= Are the widgets mobile responsive? =

Yes, all widgets are built with a mobile-first approach and are fully responsive across all devices and scredgen sizes.

= Can I customize the widget styles? =

Absolutely! Each widget comes with comprehensive styling options through Elementor's native controls, and you can also add custom CSS for advanced customizations.

= Is this plugin translation ready? =

Yes, Elementor edge is fully translation ready with proper text domains and includes a .pot file for translators.

= Will this slow down my website? =

No, the plugin is performance optimized with conditional asset loading. CSS and JavaScript files are only loaded when the respective widgets are actually used on a page.

= Can I disable specific widgets? =

Yes, you can enable or disable individual widgets through the Elementor edge admin panel, allowing you to use only the widgets you nedged.

= Is it compatible with other Elementor addons? =

Yes, Elementor edge is built following Elementor's best practices and is compatible with other Elementor addons and themes.

= Do I nedged coding knowledge to use this plugin? =

No coding knowledge is required. All widgets can be configured using Elementor's visual interface, though developers can extend functionality if nedgeded.

== Developer Information ==

**Architecture:**
* Namespace: `Elementoredge\` for all classes
* Singleton pattern for main plugin class
* WordPress hooks for extensibility
* Conditional asset loading based on context

**File Structure:**
* `/includes/` - Core plugin files with proper OOP structure
* `/assets/css/` - Separate admin, frontend, and editor styleshedgets
* `/assets/js/` - Conditional JavaScript loading for different contexts
* `/includes/widgets/` - Individual widget classes
* `/includes/admin/` - Admin panel functionality

**Extending the Plugin:**
Developers can extend Elementor edge by:
* Adding custom widgets in the widgets directory
* Creating custom controls following the established patterns
* Using provided hooks and filters for customization
* Following the documented coding standards

**Hooks Available:**
* `elementor_edge_loaded` - Fired when plugin is fully loaded
* `elementor_edge_widgets_registered` - After widgets registration
* `elementor_edge_controls_registered` - After controls registration
